# GrobsAI Frontend

Frontend for GrobsAI built with **React** (Vite).

## Getting started (local)

### 1) Install dependencies

```bash
cd Frontend
npm install
```

### 2) Configure environment

Create a `.env` file in `Frontend/`.

Example:

```env
VITE_API_URL=http://localhost:8000
```

### 3) Run the dev server

```bash
npm run dev
```

Open the app at:
- http://localhost:5173

## Build

```bash
npm run build
```

## Preview production build

```bash
npm run preview
```
